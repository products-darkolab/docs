## 2026

### March 2026

**Cohort Matching Algorithm** *(Phase 1B)*

- Learners are now automatically matched into cohorts of 4–8 peers using a skill-complementarity algorithm
- Weekly challenges rotate every Monday with track-specific prompts
- Monthly competitions launch on the first Monday of each month
- Peer code review assignments are automatically generated on lab submission
- Cohort health score tracks standup rate, review completion, and challenge activity
- Cohort leaderboard now visible in the platform sidebar

---

**Live Industry Pulse** *(Phase 1B)*

- Daily signals ingested from job postings, GitHub, arXiv, and HackerNews
- 60+ skills now have freshness scores (0–100) updated every 24 hours
- Trending and declining skills surfaced on the learning graph page
- Skills with freshness below 40 are flagged for curriculum review

---

**Four Career Tracks Launched** *(Phase 1B)*

- AI Engineer, Data Scientist, ML Engineer, and Data Engineer tracks are now live
- 271 lessons and 64 labs across all tracks
- Foundation curriculum (6 modules) auto-enrolls on registration — free for all learners
- Career tracks unlock automatically after Foundation is complete
- Track selection available from onboarding and account settings

---

**docs.safua.ai Launched**

- Public documentation site live at [docs.safua.ai](https://docs.safua.ai)
- Covers platform features, career tracks, API reference, quickstart, and FAQ

---

### February 2026

**Living Learning Graph** *(Phase 1A)*

- Skill mastery graph with 210+ skills now live for all learners
- Mastery scores update on lab completion using weighted rolling average
- Daily 2% decay reflects the real forgetting curve
- ZPD (Zone of Proximal Development) recommendations appear on dashboard
- Interactive skill graph visualisation with force-directed layout

---

**AI Tutor & Code Review** *(Phase 1A)*

- Socratic tutor available on all lesson pages — Socratic mode and Standard mode
- Code Review agent provides inline feedback on all lab submissions
- Feedback categories: correctness, style, efficiency, with line-level comments
- Tutor memory persists across sessions per learner

---

**Gamification System** *(Phase 1A)*

- XP awarded for all learning activities
- Daily streak tracking with Focus Freeze protection
- Badge system with 10 milestone badges
- Global, track, and cohort leaderboards

---

**Platform Launch** *(Phase 1A)*

- safua.ai launched with Foundation curriculum
- Monaco-powered lab editor in browser — no setup required
- FastAPI backend, Next.js frontend
- JWT authentication, Pro subscription via Stripe
