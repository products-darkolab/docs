## What a lab is

A lab is a hands-on coding exercise that runs directly in your browser. No setup required — the editor loads with starter code, instructions, and a test suite. You write your solution, submit it, and get AI feedback within seconds.

Every lab is connected to the Living Learning Graph via skill tags. Completing a lab updates your mastery scores automatically.

## The lab editor

Labs run in Monaco — the same engine that powers VS Code. You get:

- **Syntax highlighting** for Python
- **Auto-complete** and bracket matching
- **Inline AI comments** from the Code Review agent after submission
- **Test output panel** showing which tests pass

## The lab workflow

<Steps>
  <Step title="Read the instructions">
    Each lab has a brief (what you're building) and context (why it matters in production). Read both before writing code.
  </Step>
  <Step title="Study the starter code">
    Starter code includes function signatures, imports, and `TODO` comments marking where you need to write. Don't delete the structure — fill it in.
  </Step>
  <Step title="Write your solution">
    Write code that makes the tests pass. Aim for clean, readable code — not just correct code. The Code Review agent evaluates both.
  </Step>
  <Step title="Submit for review">
    Click Submit. The AI Code Review agent analyses your code and returns inline comments within seconds.
  </Step>
  <Step title="Read the feedback and iterate">
    Red glyphs are hints. Blue glyphs are Socratic questions. Green glyphs are praise. Read every comment, then decide whether to revise.
  </Step>
  <Step title="Mark complete">
    When you're satisfied with your solution, mark the lab complete. Your skill mastery scores update and XP is awarded.
  </Step>
</Steps>

<Tip>
  You can submit as many times as you want. Each submission gets a fresh AI review. Use this — professional engineers iterate constantly.
</Tip>

## Peer code review

When you submit a lab and you're in a cohort, a peer is automatically assigned to review your code within 48 hours. You'll also be assigned to review a peer's submission.

Peer reviews earn both parties XP and count toward your cohort's health score.
