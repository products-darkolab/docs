## For Learners

<Steps>
  <Step title="Create your account">
    Go to [safua.ai/register](https://safua.ai/register) and sign up with your email. You'll receive a verification email — click the link to activate your account.
  </Step>
  <Step title="Choose your career track">
    On your first login, you'll be prompted to choose a career track:
    - **AI Engineer** — RAG pipelines, agents, LLMOps
    - **Data Scientist** — statistical modelling, experimentation, ML
    - **ML Engineer** — PyTorch, distributed training, MLOps
    - **Data Engineer** — Spark, Kafka, dbt, lakehouses

    All tracks start with the shared **Foundation** (Python, SQL, Pandas, Stats, Git, Intro ML).
  </Step>
  <Step title="Complete Foundation">
    Work through the 6 Foundation modules. Each module has reading lessons and a lab. Completing Foundation unlocks your chosen career track.
  </Step>
  <Step title="Get matched to a cohort">
    Once you have some skill mastery data, the weekly matching algorithm will group you with 4–8 peers at your level. You'll receive a notification when you're matched.
  </Step>
  <Step title="Learn, build, review">
    Follow your track. Complete labs to earn XP and update your skill graph. Your AI tutor is always available — try Socratic mode for deeper understanding.
  </Step>
</Steps>

## For Developers

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/products-darkolab/safua-platform.git
    cd safua-platform
    ```
  </Step>
  <Step title="Set up the backend">
    ```bash
    cd api
    pip install -r requirements.txt
    cp .env.example .env   # fill in your keys
    python run_migrations.py
    uvicorn app.main:app --reload
    ```
    API runs at `http://localhost:8000`
  </Step>
  <Step title="Set up the frontend">
    ```bash
    cd platform
    npm install
    cp .env.local.example .env.local   # set NEXT_PUBLIC_API_URL=http://localhost:8000/v1
    npm run dev
    ```
    Frontend runs at `http://localhost:3000`
  </Step>
  <Step title="Create an admin user">
    ```bash
    cd api
    python scripts/create_admin.py --email admin@example.com --password yourpassword
    ```
  </Step>
</Steps>

See the full [Developer Setup](/developer/setup) guide for environment variables, seeding, and deployment.

## For API Users

All API endpoints are available at `https://api.safua.ai/v1`.

Authentication uses JWT Bearer tokens:

```bash
# Get a token
curl -X POST https://api.safua.ai/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "you@example.com", "password": "yourpassword"}'

# Use the token
curl https://api.safua.ai/v1/tracks/available \
  -H "Authorization: Bearer YOUR_TOKEN"
```

See the full [API Reference](/api/overview) for all 28+ endpoints.
