## Phase 0 — Foundation ✅

Core platform infrastructure.

- [x] Auth system (register, login, JWT, email verification, password reset)
- [x] User profiles and preferences
- [x] FastAPI backend with SQLAlchemy async
- [x] Next.js frontend with Tailwind v4
- [x] Railway + Vercel deployment pipeline
- [x] Stripe subscriptions (Pro + Annual, live mode)
- [x] Resend transactional email
- [x] Admin dashboard (user management, stats)
- [x] Landing page with animated code terminal, DARKO method, bento grid

## Phase 1A — Core Intelligence ✅

AI and learning intelligence features.

- [x] **Living Learning Graph** — 210 skills, mastery scoring, daily decay, ZPD recommendations, D3.js visualisation
- [x] **Multi-Agent AI Mentor** — SocraticTutorAgent, CodeReviewAgent, MotivationAgent, MetaOrchestrator
- [x] **Monaco Code Editor** — VS Code in the browser with AI inline comments (hints, questions, praise)
- [x] **Gamification v2** — Streak wagers, Focus Freeze, League system (5 tiers), spaced repetition
