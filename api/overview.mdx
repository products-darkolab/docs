## Base URL

```
https://api.safua.ai/v1
```

## Authentication

All endpoints (except `/auth/register`, `/auth/login`, and `/courses/public/stats`) require a Bearer token.

```bash
Authorization: Bearer <your_jwt_token>
```

Tokens expire after 7 days. Refresh by calling `/auth/login` again.

## Response Format

All responses are JSON. Errors follow this shape:

```json
{
  "detail": "Human-readable error message"
}
```

## Rate Limiting

- Standard endpoints: 100 requests/minute per user
- AI endpoints (`/lesson-tutor/chat`, `/code-review/submit`): 20 requests/minute per user
