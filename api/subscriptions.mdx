## Get current subscription

```bash
GET /api/v1/subscriptions/current
```

```json
{
  "data": {
    "plan": "pro",
    "status": "active",
    "current_period_end": "2026-04-07T00:00:00Z",
    "cancel_at_period_end": false
  }
}
```

## Upgrade

Upgrades are handled via [safua.ai/pricing](https://safua.ai/pricing). The API does not expose a direct upgrade endpoint — use the platform billing portal.
