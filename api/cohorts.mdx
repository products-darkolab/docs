## GET /cohorts/my-cohort

Returns the learner's active cohort, all members, and activity stats.

```bash
curl https://api.safua.ai/v1/cohorts/my-cohort \
  -H "Authorization: Bearer TOKEN"
```

**Response (not yet matched):**
```json
{
  "cohort": null,
  "message": "You haven't been matched to a cohort yet. Check back soon!"
}
```

**Response (matched):**
```json
{
  "cohort": {
    "id": "uuid",
    "name": "AI Engineer Cohort — Week 10 2026",
    "track": "ai_engineer",
    "health_score": 72.5,
    "created_at": "2026-03-02T00:00:00Z"
  },
  "members": [...],
  "member_count": 6
}
```

    "ends_at": "2026-03-14T00:00:00Z",
    "submitted": false,
    "days_remaining": 5
  }

## POST /cohorts/challenges/{challenge_id}/submit

Submit a solution to a challenge.

```bash
curl -X POST https://api.safua.ai/v1/cohorts/challenges/UUID/submit \
  -H "Authorization: Bearer TOKEN" \
  -d '{"code": "def rag_pipeline():\n    ...", "notes": "Used pgvector for storage"}'
```

**Response:**
```json
{
  "message": "Submitted! +150 XP awarded.",
  "xp_awarded": 150
}
```

---

## POST /cohorts/peer-reviews/{assignment_id}/complete

Submit peer review feedback. Requires at least 20 characters of feedback and a score 0–100.

```bash
curl -X POST https://api.safua.ai/v1/cohorts/peer-reviews/UUID/complete \
  -H "Authorization: Bearer TOKEN" \
  -d '{
    "feedback": "Great use of list comprehensions. Consider extracting the embedding logic into a separate function for reusability.",
    "score": 82
  }'
```

**Response:**
```json
{
  "message": "Peer review submitted! +50 XP earned.",
  "xp_awarded": 50,
  "score_given": 82
}
```

## Admin Endpoints

### GET /cohorts/admin/all
All cohorts ranked by health score. Flags cohorts with health < 40 as `needs_attention`. **Admin only.**

### POST /cohorts/run-matching
Triggers cohort matching in the background. Optionally scope to a specific track.

```bash
curl -X POST https://api.safua.ai/v1/cohorts/run-matching \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -d '{"track": "ai_engineer"}'
```
