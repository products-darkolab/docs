## POST /code-review/submit

Submit lab code. Returns structured AI review and triggers peer review assignment if the user is in a cohort.

See the [AI Agent](/api/agent) page for the full request/response schema.

## GET /code-review/submissions

Returns all of the user's past lab submissions with their AI review scores.

```json
[
  {
    "id": "uuid",
    "lab_id": "uuid",
    "lab_title": "Build a RAG Pipeline",
    "overall_score": 82,
    "submitted_at": "2026-03-07T14:22:00Z",
    "peer_review_status": "pending"
  }
]
```

## GET /code-review/submissions/{submission_id}

Full submission detail including the complete AI review JSON and peer review feedback (if completed).
