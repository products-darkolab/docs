## List all tracks

```bash
GET /api/v1/tracks
```

```json
      "id": 1,
      "slug": "foundation",
      "title": "Foundation",
      "is_foundation": true,

      "module_count": 6

      "is_foundation": false,
      "estimated_weeks": 22,
      "level_count": 5,
      "module_count": 17

  ]
}
```

## Get a specific track

```bash
GET /api/v1/tracks/{slug}
```

```bash
GET /api/v1/tracks/ai-engineer
```

## Get track progress for the authenticated user

```bash
GET /api/v1/tracks/{slug}/progress
```

```json
{
  "data": {
    "track_slug": "ai-engineer",
    "is_unlocked": true,
    "levels_completed": 2,
    "modules_completed": 8,
    "lessons_completed": 34,
    "labs_completed": 8,
    "overall_percentage": 47.2
  }
}
```
