## The curriculum structure

Every track follows the same structure:

```
Foundation (shared, ~7 weeks)
└── Career Track (~20–22 weeks)
    ├── Level 1 — Core Fundamentals (free)
    ├── Level 2 — Applied Skills
    ├── Level 3 — Production Patterns
    ├── Level 4 — Advanced Systems
    └── Level 5 — Capstone
```

Foundation is completed once — it unlocks your chosen career track automatically.

Each level contains multiple modules. Each module contains 3–6 lessons and 1–3 labs. Labs connect to the Living Learning Graph via skill tags.
