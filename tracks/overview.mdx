## How Tracks Work

Every learner on Safua starts with the **Foundation track** — 6 core modules shared across all four career paths. Once Foundation is complete, your chosen career track unlocks automatically.

```
Foundation (all learners)
├── Python Fundamentals
├── SQL & Databases
├── Data Manipulation with Pandas
├── Statistics for ML
├── Git & Collaboration
└── Introduction to ML
        │
        ▼ (unlock on completion)
┌───────────────────────────────────────────────┐
│  Choose one:                                  │
│  AI Engineer │ Data Scientist │ ML Engineer   │
│              │ Data Engineer                  │
└───────────────────────────────────────────────┘
```

## Track Comparison

| | AI Engineer | Data Scientist | ML Engineer | Data Engineer |
|---|---|---|---|---|
| Duration | ~22 weeks | ~20 weeks | ~22 weeks | ~20 weeks |
| Levels | 5 | 5 | 5 | 5 |
| Modules | 17 | 14 | 12 | 15 |
| Focus | LLMs, RAG, Agents | Stats, Experimentation | PyTorch, MLOps | Pipelines, Infra |
| Capstone | AI Product | Kaggle Competition | ML System | Data Platform |

## Foundation Track (~7 weeks)

Shared by all learners. **Must be completed before any career track unlocks.**

| Module | Key Skills |
|---|---|
| Python Fundamentals | Functions, OOP, async/await, testing |
| SQL & Databases | Joins, window functions, CTEs, indexes |
| Data Manipulation with Pandas | DataFrames, groupby, merges, cleaning |
| Statistics for ML | Distributions, hypothesis testing, Bayesian basics |
| Git & Collaboration | Branching, PRs, conflict resolution, CI/CD |
| Introduction to ML | Supervised/unsupervised, model evaluation, bias-variance |

## Cohort Matching by Track

After you've completed some of Foundation, the weekly cohort matching algorithm groups you with 4–8 learners on the same track. Your cohort has:

- **Weekly challenges** — track-specific coding challenges with XP rewards
- **Monthly competitions** — a capstone-style challenge judged by peers
- **Peer code reviews** — structured feedback on each other's lab submissions
- **Standup feed** — async daily updates to stay accountable
- **Weekly leaderboard** — friendly XP competition within your cohort
