## Overview

Every Safua learner starts here, regardless of which career track they choose. Foundation builds the core skills that all AI and data roles share — Python, SQL, Pandas, statistics, Git, and an intro to ML.

**Required by:** All tracks
**Duration:** ~7 weeks
**Modules:** 6
**Labs:** 6

## Curriculum

| Module | Key Skills | Labs |
|---|---|---|
| Python Fundamentals | Functions, OOP, comprehensions, async/await, testing | Build a data processing pipeline with error handling and tests |
| SQL & Databases | SELECT, JOINs, window functions, CTEs, indexes | Analyse a real dataset — answer 10 business questions with SQL |
| Data Manipulation with Pandas | Series, DataFrames, groupby, merges, cleaning, reshape | Clean a messy real-world dataset from raw CSV to analysis-ready |
| Statistics for ML | Distributions, central limit theorem, hypothesis testing, Bayes | Run a complete A/B test analysis on a provided dataset |
| Git & Collaboration | Branching, merging, pull requests, conflicts, CI basics | Collaborate on a shared repo: branch, PR, review, merge |
| Introduction to ML | Supervised/unsupervised, train/val/test split, evaluation metrics | Train, evaluate, and explain a classification model |

## What Happens After Foundation?

When you complete all 6 Foundation modules, your chosen career track unlocks automatically. You'll see a notification and the track content becomes available on your courses page.

<Info>
Your Foundation enrollment stays active after completion. You can revisit any Foundation module at any time — skills decay if not reviewed, and the spaced repetition system will remind you when a concept needs refreshing.
</Info>
