## Overview

The ML Engineer track trains you to build, train, and deploy machine learning systems at scale. You'll work in PyTorch, build data pipelines, serve models under production load, and set up monitoring to catch issues before they affect users.

**Prerequisites:** Foundation track completed
**Duration:** ~22 weeks | **Labs:** 12 | **Capstone:** Production ML system design

## Curriculum

### Level 1 — Deep Learning with PyTorch (~5 weeks)
PyTorch tensors and autograd, building neural networks, training loops (early stopping, mixed precision, gradient clipping), CNNs, transfer learning.

### Level 2 — Distributed Training (~4 weeks)
DataLoaders and data augmentation, multi-GPU training with `torch.distributed`, gradient accumulation, DeepSpeed basics, training efficiency profiling.

### Level 3 — Data Pipelines for ML (~4 weeks)
Feature stores, data versioning with DVC, building reproducible pipelines, handling data drift at ingestion, Great Expectations for data quality.

### Level 4 — MLOps (~6 weeks)
Experiment tracking with MLflow, CI/CD for models with GitHub Actions, model serving with BentoML (target p99 < 100ms), A/B model deployment, drift detection with Evidently, alerting.

### Level 5 — Capstone (~3 weeks)
Design and implement a production-ready ML system: training pipeline + serving layer + monitoring dashboard. Peer reviewed. **+500 XP**.
