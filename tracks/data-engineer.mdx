## Overview

The Data Engineer track trains you to build the infrastructure that powers every data-driven product. You'll move from data modelling and warehousing, through transformations with dbt, to streaming pipelines and lakehouse architectures.

**Prerequisites:** Foundation track completed
**Duration:** ~20 weeks | **Labs:** 15 | **Capstone:** Full data platform build

## Curriculum

### Level 1 — Data Warehousing (~4 weeks)
Dimensional modelling (Kimball), star schemas, slowly changing dimensions, Snowflake/BigQuery/Redshift fundamentals, query optimisation, partitioning.

### Level 2 — dbt & SQL Transformations (~4 weeks)
dbt project structure, staging/intermediate/mart layers, tests and documentation, incremental models, dbt Cloud, Jinja templating.

### Level 3 — Distributed Processing with Spark (~5 weeks)
PySpark fundamentals, DataFrames API, optimisation (partitioning, caching, broadcast joins), fixing data skew, Spark Structured Streaming.

### Level 4 — Streaming & Event-Driven Pipelines (~4 weeks)
Kafka fundamentals, producers and consumers, exactly-once semantics, Kafka Streams, Flink basics, real-time aggregations.

### Level 5 — Capstone (~3 weeks)
Build a complete mini data platform: ingest → dbt transform → Airflow orchestration → data quality checks → monitoring dashboard. Peer reviewed. **+500 XP**.
